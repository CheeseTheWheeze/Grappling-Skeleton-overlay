FightAI Portable
=================

1) Run the bootstrap script to download the embedded runtime:
   FightAI\tools\bootstrap_python.ps1

2) Ensure the embedded Python runtime exists at:
   FightAI\runtime\python\python.exe

3) Launch the app by double-clicking FightAI.cmd.

If you supply a FightAI.exe launcher, it can be used instead of FightAI.cmd.
